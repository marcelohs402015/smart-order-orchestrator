# Imagens do Frontend

Esta pasta contém screenshots e imagens do frontend do Smart Order Orchestrator.

## Imagens Disponíveis

- `dashboard.png` - Screenshot do dashboard principal mostrando métricas e ações rápidas

## Como Adicionar Novas Imagens

1. Adicione a imagem nesta pasta
2. Atualize o README.md do frontend com a referência
3. Use formato PNG ou JPG para screenshots
4. Mantenha nomes descritivos e em minúsculas
